# YouTube Video Downloader 🎬

A simple command-line YouTube video downloader built with Python and yt-dlp.

## Features
- Download any YouTube video as MP4
- Live download progress and speed
- Save to any folder you choose
- Download multiple videos in one session
- Handles invalid URLs gracefully

## Requirements
- Python 3.x
- yt-dlp

## Installation

1. Clone the repository:
   git clone https://github.com/Sacred2609/youtube-downloader.git

2. Install the required library:
   pip install yt-dlp

## Usage
Run the script:
   python main.py

Then follow the prompts to enter a URL and save location.
